CRON(1) - General Commands Manual

# NAME

**cron** - clock daemon

# SYNOPSIS

**cron**
\[**-f**&nbsp;*file*]
\[**-n**]

# DESCRIPTION

**cron**
schedules commands to be run at specified dates and times.

# OPTIONS

**-f** *file*

> Use the specified
> *file*
> instead of the default
> */etc/crontab*.

**-n**

> Do not daemonize.
