CAT(1) - General Commands Manual

# NAME

**cat** - concatenate files

# SYNOPSIS

**cat**
\[**-u**]
\[*file&nbsp;...*]

# DESCRIPTION

**cat**
reads each
*file*
in sequence and writes it to stdout.
If no
*file*
is given
**cat**
reads from stdin.

# OPTIONS

**-u**

> Unbuffered output.

# STANDARDS

The
**cat**
utility is compliant with the
specification.
