HOSTNAME(1) - General Commands Manual

# NAME

**hostname** - set or print host name

# SYNOPSIS

**hostname**
\[*name*]

# DESCRIPTION

**hostname**
sets the current host name to
*name*.
If no
*name*
is given, the current host name is written to stdout.
